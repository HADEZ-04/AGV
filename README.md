# Problem Statement:
An industry-sponsored laboratory in our college was setup with multiple workstations, each station performs a specific operation on a workpiece,the transportation of the worpiece was done manually.The absence of an automated material handling system limited workflow continuity and reduces overall productivity in the multi-station assembly setup.
# Solution:
Developed a line-follower Autonomous Guided Vehicle (AGV) to automate the transportation of workpieces between different workstations and inventory within the lab. The system uses a joystick-based wireless control subsystem that enables operators to call and dispatch the AGV between stations, with communication implemented using Wi-Fi via the ESP-NOW protocol. This solution improves material flow efficiency, reduces manual effort, and enables smoother coordination across multiple assembly stations.
